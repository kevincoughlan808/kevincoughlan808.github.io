# Kevin Coughlan's Blog

A simple Jekyll blog to journal my progress, hosted on GitHub Pages.

## Local Development

1. Install Ruby and Bundler.
2. Clone this repository.
3. Run `bundle install` to install dependencies.
4. Run `bundle exec jekyll serve` to start the local server.
5. Open `http://localhost:4000` in your browser.

## Deployment

Push changes to the `main` branch. GitHub Pages will automatically build and deploy the site.

## Writing Posts

Add new posts to the `_posts` directory with the format `YYYY-MM-DD-title.md`.

Example:

---
layout: post
title: "My Post Title"
date: 2026-03-13
---

Post content here.
